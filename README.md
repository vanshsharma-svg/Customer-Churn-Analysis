# 📊 Customer Churn Analysis

## 📌 Project Overview

This is a beginner-friendly **Customer Churn Analysis** project built using **Python, Pandas, NumPy, SQLite, Matplotlib, and Seaborn**.

The project was created for **learning and practice** using a **small sample dataset** to simulate a real-world customer churn analysis workflow.

It covers the complete analytics process, including **data cleaning, feature engineering, KPI calculation, exploratory data analysis (EDA), data visualization, and business insights.**

---

## 🎯 Objectives

- Clean and preprocess customer data
- Calculate important business KPIs
- Analyze customer churn behavior
- Visualize customer trends
- Generate business insights and recommendations
